# Magic Tower Demo

一个基于原始项目继续整理和扩展的 H5 魔塔 Demo，目前已经可以直接游玩，并补充了一些更适合试玩和演示的功能。

## 当前状态

- 当前为 Demo 版本
- 可游玩楼层：第 1 层到第 12 层
- 到达最高层时会提示“后续内容开发中”
- 支持触屏操作，也支持键盘快捷键

## 本次整理后的功能

- `W / A / S / D` 或方向键移动
- `M` 打开菜单
- `B` 打开怪物图鉴
- `T` 打开楼层传送
- `G` 打开金手指面板
- `K / L` 快速闪现到上一层 / 下一层楼梯边
- 支持自定义主角形象
- 支持背景音乐自动播放
- 支持怪物图鉴、商店、楼层切换
- 支持直接修改角色属性与钥匙数量

## 运行方式

### 1. 安装依赖

```bash
npm install
```

### 2. 监听构建

```bash
npm run watch
```

### 3. 打开页面

构建产物会输出到：

```bash
dist/view/index.html
```

你可以用任意静态服务器打开 `dist` 目录，或者直接在本地浏览器中访问构建后的页面。

## 快捷键

| 功能 | 按键 |
| --- | --- |
| 移动 | `W / A / S / D` 或方向键 |
| 菜单 | `M` |
| 怪物图鉴 | `B` |
| 传送 | `T` |
| 金手指 | `G` |
| 上一层闪现 | `K` |
| 下一层闪现 | `L` |
| 关闭弹层 | `Esc` |
| 对话继续 / 确认 | `Enter / Space` |

## 项目结构

```text
src/
├── audio/           背景音乐
├── images/          角色、怪物、UI、物品资源
├── modules/         游戏核心逻辑
├── index.html       页面模板
├── index.js         入口逻辑
└── style.css        样式

editor/              地图编辑器相关内容
animation/           动画实验/素材
webpack.config.js    构建配置
```

## 说明

- 原始项目使用传统 Webpack 构建，代码结构保持了较多早期前端项目风格
- 当前仓库主要定位为“可试玩 Demo + 后续扩展基础”
- 如果你想继续扩关、补剧情或增加更多楼层，可以优先从 `src/modules/Maps.js` 和 `src/modules/Events.js` 开始
